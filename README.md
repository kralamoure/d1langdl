# d1langdl

`d1langdl` is a program for downloading Dofus 1 lang files.

## Requirements

- [Git](https://git-scm.com/)
- [Go](https://golang.org/)

## Build

```sh
git clone https://github.com/kralamoure/d1langdl
cd d1langdl
go build
```

## Installation

```sh
go get -u -v github.com/kralamoure/d1langdl
```

## Usage

```sh
d1langdl --help
```
